Instructions

  1. Click on plus icon to add a Sticky note
  2. Click and drag to move sticky note
  3. Click on delete icon to delete selected sticky note(NOTE: if no sticky note is selected, lastly added sticky note will deleted)
  4. Click on delete all icon to delete all Sticky note
  5. Click on Palette icon to change the background colour of selected sticky note
  6. Click on Text colour icon to change the text colour of sticky note
  7. Click on Secondary icon to switch between dark mode and light mode
